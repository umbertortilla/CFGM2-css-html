Apache Tomcat (antes bajo el proyecto Apache Yakarta) es un contenedor de servlets desarrollado en Apache Software Foundation. Tomcat implementa las especificaciones de servlet y JavaServer Pages (JSP) de Sun Microsystems, proporcionando un entorno para el código Java a ejecutar en cooperación con un servidor web. Éste añade herramientas para la configuración y el mantenimiento, pero también puede ser configurado editando los archivos de configuración que normalmente están en formato XML. Tomcat incluye su propio servidor HTTP, por eso también se le considera un servidor web independiente.
